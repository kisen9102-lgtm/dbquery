功能简介

本平台支持多种数据库的在线查询与管理，根据角色不同可使用以下功能：

数据库查询
按数据库名或 IP + 端口跨实例检索，快速定位数据库所在实例。
SQL 查询
在线编辑器支持 MySQL / TiDB / PostgreSQL / Redis / MongoDB，可浏览对象、多语句执行并导出结果。
实例管理
注册、编辑、删除各类数据库实例，支持测试连接。仅 admin / root 可操作。
用户管理
创建用户并分配角色（root / admin / query），修改密码。仅 admin / root 可操作。
用户组管理
将实例与用户组绑定，控制 query 用户可见的实例范围。仅 admin / root 可操作。
角色说明
角色 权限说明
root 全部权限，不受任何限制。
admin 可管理实例、用户、用户组；不可修改 root 账号。
query 只读查询；只能看到所属用户组内的实例；实例 IP / 端口不可见。
